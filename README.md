# Electricity-Usage-Analysis
Electricity Consumption Analysis (Tableau + Flask)

This project analyzes electricity consumption patterns across different states in India using interactive data visualizations built in Tableau and presented through a Flask web application.

The goal of the project is to explore electricity usage trends across states and identify patterns in energy consumption through interactive dashboards.

LIVE URL - https://electricity-consumption-0ldd.onrender.com/

Video Demonstration - https://drive.google.com/file/d/157MzOJfnVMy8lKFfrW3L8xYH-RRj1lqb/view?usp=sharing

Project Overview The analysis focuses on understanding electricity consumption trends across Indian states and presenting insights using interactive dashboards and a data story.

Visualizations were created using Tableau Public and embedded into a Flask-based website for easy access and navigation.

Users can explore:

• Interactive dashboards for electricity consumption analysis • A structured data story explaining insights • Visualizations comparing consumption across states • Summary findings from the dataset

Key Insights

• Maharashtra shows the highest electricity consumption among Indian states • Gujarat ranks as the second highest electricity consuming state • Sikkim records the lowest electricity consumption • Electricity consumption varies across different regions of India
